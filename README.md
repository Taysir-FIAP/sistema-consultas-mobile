# 📱 Sistema de Consultas Mobile

Aplicação mobile desenvolvida com **React Native + TypeScript**, com o objetivo de gerenciar e consultar informações de forma simples e eficiente.

## 🚀 Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- JavaScript (ES6+)

## 📂 Estrutura do Projeto
sistema-consultas-mobile/
├── assets/ # Imagens, ícones e recursos estáticos
├── src/ # Código-fonte principal da aplicação
├── App.tsx # Componente raiz da aplicação
├── index.ts # Ponto de entrada
├── app.json # Configurações do Expo
├── package.json # Dependências e scripts
├── tsconfig.json # Configuração do TypeScript
└── .gitignore # Arquivos ignorados pelo Git


## 📦 Instalação

Clone o repositório:

bash
git clone https://github.com/Taysir-FIAP/sistema-consultas-mobile.git

Acesse a pasta do projeto:

cd sistema-consultas-mobile

Instale as dependências:

npm install

ou

yarn
▶️ Como executar

Inicie o projeto com:

npx expo start

Depois, você pode:

Rodar no emulador Android/iOS
Usar o app Expo Go no celular
Abrir no navegador (modo web)
📱 ##Funcionalidades
